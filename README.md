# Neo IPTV Player

iptv-org의 공개 M3U 플레이리스트를 활용하여 브라우저에서 편리하게 실시간 방송 채널을 시청할 수 있는 세련되고 현대적인 웹 IPTV 플레이어입니다.

## 주요 기능
- **다국어 및 다양한 플레이리스트 지원**: 기본 한국 채널 및 뉴스, 스포츠, 영화 등 다양한 글로벌 플레이리스트 전환 기능 제공.
- **실시간 검색**: 로드된 채널 목록에서 원하는 채널을 키워드로 실시간 필터링.
- **사용자 정의 주소**: 외부 M3U 플레이리스트 주소를 직접 입력하여 시청 가능.
- **Modern UI**: 다크 모드 및 Glassmorphism 스타일을 적용한 미려한 사용자 인터페이스.
- **HLS 재생**: HLS.js 라이브러리를 사용하여 `.m3u8` 라이브 공용 채널 스트리밍 지원.

## 실행 방법
1. 이 저장소를 다운로드하거나 클론합니다.
2. `index.html` 파일을 더블 클릭하여 아무 브라우저에서나 즉시 실행할 수 있습니다.
